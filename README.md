# s-h-f-serv00
sing-box + hysteria2 + freebsd 支持在 serv00 上搭建 3 个 hysteria2 节点

# 注意
> 此脚本会清除 serv00 上的全部端口，并按照用户输入的生成 hy2 个数生成全新的 udp 端口
> 由于端口限制你最多只能生成3个节点，这只是个测试脚本，写的很low，你可以魔改爆改本脚本
